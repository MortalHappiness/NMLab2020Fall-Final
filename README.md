## Prerequsite

* NodeJS
* Truffle
* Ganache

## Quick Start

Make sure you have opened ganache and then type the following commands in the terminal.

```shell
$ npm install
$ truffle compile
$ truffle migrate
```

In another terminal

```shell
$ cd client
$ npm install
$ npm start
```
